# Лабораторная работа №1 

## Описание проекта
В рамках лабораторной работы был реализован веб-сервис для управления и монетизации видеоконтента (аналог YouTube), основанный на бизнес-процессе, описанном с использованием нотации **BPMN 2.0**.

## Основные этапы работы
- Проанализирован и выбран бизнес-процесс (управление видеоконтентом)
- Построена модель процесса в нотации **BPMN 2.0** (см. диаграммы на стр. 2–3) :contentReference[oaicite:0]{index=0}
- Спроектирована структура базы данных
- Разработано backend-приложение на **Spring Boot**
- Реализован **REST API** для взаимодействия с системой (пример запросов на стр. 4–10) :contentReference[oaicite:1]{index=1}
- Настроено хранение данных в **PostgreSQL**
- Подготовлены тестовые запросы (Postman/аналогичные инструменты)
- Выполнено развертывание приложения

## Функциональность
- загрузка и управление видеоконтентом
- модерация видео
- настройка доступа и публикации
- обработка пользовательских действий
- взаимодействие через REST API

## Используемые технологии
- Java
- Spring Boot
- PostgreSQL
- REST API
- BPMN 2.0

## Структура проекта
- бизнес-процесс (BPMN диаграммы)
- backend (Spring Boot приложение)
- база данных (ER-диаграмма)
- API (эндпоинты и тесты)

2. Настроить параметры подключения в `application.properties`
3. Запустить приложение:
