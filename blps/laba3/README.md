# Лабораторная работа №2 

## Описание проекта
В данной работе было доработано приложение из лабораторной №1, реализующее систему управления видеоконтентом. Основной упор сделан на **управление транзакциями** и **разграничение доступа пользователей**.

## Основные задачи
- Расширение бизнес-процесса и его реализация (см. BPMN диаграммы на стр. 2–3) :contentReference[oaicite:0]{index=0}
- Реализация управления транзакциями
- Настройка системы авторизации и разграничения доступа
- Доработка REST API

## Реализованный функционал

### Управление транзакциями
- Использован **Spring JTA**
- Применен менеджер транзакций **Narayana**
- Объединены взаимосвязанные операции в единые транзакции
- Реализовано программное управление транзакциями

### Безопасность и доступ
- Настроен **Spring Security**
- Реализована модель ролей и прав доступа
- Добавлена аутентификация через **JWT**
- Пользователи хранятся в реляционной базе данных

### REST API
- Расширены и доработаны существующие эндпоинты
- Добавлены защищенные методы
- Подготовлены тестовые запросы (см. примеры на стр. 5–6) :contentReference[oaicite:1]{index=1}

## Используемые технологии
- Java
