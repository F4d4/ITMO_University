## Лабораторная работа №3

В рамках работы был разработан сценарий сборки проекта с помощью **Apache Ant**.

### Основные задачи:
- Настроена автоматическая компиляция исходного кода
- Реализована сборка проекта в исполняемый `jar`-архив
- Добавлен `MANIFEST.MF` с версией проекта и главным запускаемым классом
- Все переменные и константы вынесены в отдельный файл параметров
- Реализованы отдельные цели Ant:
  - `compile` — компиляция проекта
  - `build` — сборка и упаковка в `jar`
  - `clean` — очистка временных и скомпилированных файлов
  - `test` — запуск JUnit-тестов
  - `xml` — валидация XML-файлов
  - `history` — поиск последней рабочей версии проекта в Git
- Добавлена возможность автоматически откатываться по истории Git при ошибке компиляции
- Формируется diff-файл для изменений, следующих после последней успешно собираемой ревизии

Работа позволила автоматизировать процесс сборки, тестирования и проверки проекта, а также закрепить навыки работы с Apache Ant и Git.
