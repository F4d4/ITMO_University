## Лабораторная работа №2

В рамках работы были изучены и применены системы контроля версий **SVN и Git**.

### Основные задачи:
- Настроены локальные репозитории SVN и Git
- Выполнена инициализация и загрузка начальной ревизии проекта
- Воспроизведена сложная последовательность операций над кодом по блок-схеме:
  - создание веток
  - выполнение коммитов разными пользователями
  - переключение между ветками
  - слияние (merge) веток
- Реализовано автоматическое разрешение конфликтов при слиянии
- Написаны bash-скрипты для автоматизации работы с SVN и Git
- Построены графы истории коммитов (SVN и Git)

Работа позволила закрепить навыки работы с ветвлением, слиянием и управлением историей изменений в системах контроля версий. :contentReference[oaicite:0]{index=0}
