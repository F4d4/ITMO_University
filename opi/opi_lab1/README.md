## Лабораторная работа №1

В рамках работы был разработан документ **Software Requirements Specification (SRS)** для веб-приложения — финансового маркетплейса (аналог Банки.ру).

### Основные задачи:
- Проведен анализ предметной области и целевой аудитории
- Сформулированы **функциональные и нефункциональные требования**
- Описаны пользовательские роли: гость, пользователь, администратор, редактор
- Спроектированы основные возможности системы:
  - поиск и сравнение финансовых продуктов
  - регистрация и авторизация пользователей
  - оформление заявок и консультаций
  - отзывы, комментарии и персональный кабинет
- Определены требования к:
  - производительности и надежности
  - безопасности и защите данных
  - UI/UX и удобству использования
- Выбраны технологии (HTML, CSS, JS, PHP, SQL, Nginx)
- Разработаны **UML Use Case диаграммы** и сценарии использования системы
